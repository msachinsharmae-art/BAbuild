### What I've Learned About AI

Through building real tools with Claude Code and the Claude API, here's what I've figured out:

- **Prompt engineering is system design** — defining 30+ tools for an AI agent is closer to architecture than coding
- **You don't need to be a developer to build software** — you need to understand how systems work, then instruct AI precisely
- **Agentic AI changes the game for BAs** — I can prototype a working payroll engine, not just write requirements for one
- **Domain knowledge is the real moat** — AI can write code, but it needs someone who understands payroll tax rules, compliance logic, and business workflows to build the right thing
- **Test automation is the fastest win** — described test scenarios to Claude Code, got 50+ working Playwright specs
