## My Approach to AI & Prompt Engineering
I treat AI as a collaborator for thinking, drafting, and exploring — not a replacement for engineering.

- Write structured prompts: explicit role, constraints, inputs, verification steps
- Use AI to accelerate the *first 70%* (drafts, scaffolds, research synthesis) — final judgement stays with me
- Validate every AI-assisted output against source-of-truth: real APIs, real data, real stakeholder feedback
- Prefer small, inspectable prototypes over opaque automations
